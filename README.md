# Online Shopping Customer Behavior Study (Python)

## Project Overview
This project analyzes online shopping customer behavior to understand purchase patterns.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of purchase behavior
- Logistic Regression model for purchase prediction

## Dataset
Online Shoppers Purchasing Intention Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
